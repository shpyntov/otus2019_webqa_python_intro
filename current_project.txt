Тестируемый проект - вэб-приложение, система обновления ПО камер видеонаблюдения.
Front-end на React JS. Back-end на Go. Для хранения данных используются Postgres и Reindexer.

Атотесты есть только для основного функционала.
UI-тестами покрыто около 20% функционала, пишут тестировщики.
Для API около 30%, пишут тестировщики.
Интеграционных тестов нет.
Unit-тесты пишут разработчики.

На проекте явно не хватает автотестов для ui и api для более быстрого прохождения регресса и ускорения релизов.
